Dominio de Datos (SQL)

No solo conecto la API, también diseño arquitecturas de bases de datos para trackear inventario y ventas históricas.

  • ✓ Optimización de queries para reportes de facturación masiva.
  • ✓ Modelado de datos para sistemas Multi-cuenta. 
-- Reporte de Rentabilidad
SELECT 
    item_id, 
    SUM(sale_price - cost) AS net_profit
FROM sales_data
WHERE status = 'delivered'
GROUP BY item_id
ORDER BY net_profit DESC;
